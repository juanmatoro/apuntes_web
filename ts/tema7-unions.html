Union types Intersecciones Unions discriminadas

1Union types

Una union permite varios tipos posibles:

type ID = string | number;

function buscar(id: ID) {
  return `Buscando ${id}`;
}

2Intersecciones

Combinan propiedades de multiples tipos:

type Base = { id: string };
type Auditable = { createdAt: Date };
type Entidad = Base & Auditable;

const e: Entidad = { id: "1", createdAt: new Date() };

3Unions discriminadas

Patron clave para modelar estados de UI o respuestas de API.

type Ok = { status: "ok"; data: string[] };
type Err = { status: "error"; message: string };
type Resultado = Ok | Err;

function render(r: Resultado): string {
  if (r.status === "ok") return r.data.join(", ");
  return `Error: ${r.message}`;
}
