Tipos mapeados Transformar objetos Funciones Objetos nativos

1Utility types mas usados

interface Usuario {
  id: string;
  nombre: string;
  email?: string;
}

type UsuarioParcial = Partial<Usuario>;
type UsuarioCompleto = Required<Usuario>;

2Pick, Omit y Record

type UsuarioPublico = Pick<Usuario, "id" | "nombre">;
type UsuarioInterno = Omit<Usuario, "email">;

type Roles = "admin" | "editor" | "viewer";
type Permisos = Record<Roles, boolean>;

3Utilidades para funciones

function crearSesion(userId: string, remember: boolean) {
  return { token: "abc", userId, remember };
}

type ParamsSesion = Parameters<typeof crearSesion>;
type ResultadoSesion = ReturnType<typeof crearSesion>;

Estas utilidades reducen duplicacion de tipos y evitan desalineaciones.

4Repaso de objetos nativos de JS/TS

  • Wrappers: String, Number, Boolean (usar primitivos por defecto).
  • Arrays: metodos mutables e inmutables (push, map, filter, reduce).
  • Estructuras: Set (unicidad) y Map (clave-valor robusto).
  • Utilidades no instanciables: Math y JSON.
const ids = new Set<string>(["a", "b", "a"]); // a, b
const precios = new Map<string, number>([["p1", 10]]);

const raw = JSON.stringify({ ok: true });
const parsed = JSON.parse(raw) as { ok: boolean };
const redondeado = Math.round(10.4);
