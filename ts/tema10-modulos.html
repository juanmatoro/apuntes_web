Modulos ES Barrels Namespaces

1Modulos ES

// usuario.ts
export interface Usuario { id: string; nombre: string; }
export const crearUsuario = (id: string, nombre: string): Usuario => ({ id, nombre });

// app.ts
import { crearUsuario } from "./usuario";
const u = crearUsuario("u1", "Ana");

Usa siempre import/export, especialmente con bundlers modernos.

2Barrel files

Un archivo index.ts puede reexportar simbolos para simplificar imports.

// dominio/index.ts
export * from "./usuario";
export * from "./pedido";

3Namespaces

Son utiles en casos legacy o sin sistema de modulos, pero en proyectos modernos se prefieren modulos ES.

namespace MathExtra {
  export const doble = (n: number) => n * 2;
}
