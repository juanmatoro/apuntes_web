1Clases tipadas

class Producto {
  constructor(
    public sku: string,
    public nombre: string,
    private precio: number
  ) {}

  getPrecio(): number {
    return this.precio;
  }
}

2Modificadores de acceso

  • public: visible en cualquier lugar.
  • private: visible solo dentro de la clase.
  • protected: visible en clase y subclases.
  • readonly: asignable solo en declaracion o constructor.

3Herencia e interfaces

interface Imprimible {
  imprimir(): string;
}

class UsuarioBase {
  constructor(protected id: string) {}
}

class Admin extends UsuarioBase implements Imprimible {
  imprimir(): string {
    return `Admin ${this.id}`;
  }
}

Prioriza composicion sobre herencia profunda para mantener bajo acoplamiento.

4Objetos, funciones constructoras y prototipos

Las clases de ES6 son azucar sintactico sobre prototipos. Entender la cadena de prototipos ayuda a depurar y modelar mejor.

function Persona(this: any, nombre: string) {
  this.nombre = nombre;
}
(Persona as any).prototype.saludar = function () {
  return `Hola ${this.nombre}`;
};

5Accessors: getters y setters

class Cuenta {
  private _saldo = 0;

  get saldo(): number {
    return this._saldo;
  }

  set saldo(valor: number) {
    if (valor < 0) throw new Error("Saldo invalido");
    this._saldo = valor;
  }
}
