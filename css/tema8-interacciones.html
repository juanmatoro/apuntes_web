1Focus

Esencial para accesibilidad y navegación por teclado.

Pseudo-clases de focus

  • :focus - Elemento con foco (cualquier método)
  • :focus-visible - Solo muestra outline cuando es necesario
  • :focus-within - Padre contiene elemento con foco
/* Quitar outline por defecto (no recomendado) */
button:focus {
    outline: none;
}

/* Mejor: usar focus-visible */
button:focus-visible {
    outline: 2px solid blue;
    outline-offset: 2px;
}

/* Estilizar formulario cuando tiene foco interno */
form:focus-within {
    box-shadow: 0 0 0 3px rgba(59, 130, 246, 0.3);
}

Outline

.element:focus {
    outline: 2px solid #3b82f6;
    outline-offset: 2px;  /* Espacio entre borde y outline */
}

2Overflow

Controla qué pasa cuando el contenido excede el contenedor.

Valores

  • visible - Contenido se desborda (default)
  • hidden - Recorta el contenido
  • scroll - Siempre muestra scrollbar
  • auto - Scrollbar solo si es necesario
  • clip - Como hidden, pero sin crear BFC
.container {
    overflow: auto;           /* Ambos ejes */
    overflow-x: auto;         /* Solo horizontal */
    overflow-y: scroll;       /* Solo vertical */

    /* Propiedades lógicas */
    overflow-inline: auto;
    overflow-block: hidden;
}

text-overflow

.truncate {
    white-space: nowrap;
    overflow: hidden;
    text-overflow: ellipsis;  /* ... al final */
}

3Comportamiento de scroll

scroll-behavior

Suaviza el scroll al navegar con anclas.

html {
    scroll-behavior: smooth;
}

/* O en un contenedor específico */
.scroll-container {
    scroll-behavior: smooth;
}

overscroll-behavior

Controla qué pasa al llegar al límite del scroll.

.modal {
    overscroll-behavior: contain;
    /* Evita que el scroll "escape" al body */
}

/* Valores */
overscroll-behavior: auto;     /* Default */
overscroll-behavior: contain;  /* No propaga */
overscroll-behavior: none;     /* Sin efecto bounce */

scrollbar-gutter (2022)

Reserva espacio para la scrollbar.

.container {
    scrollbar-gutter: stable;
    /* Evita saltos cuando aparece/desaparece scrollbar */

    scrollbar-gutter: stable both-edges;
    /* Espacio en ambos lados */
}

scroll-padding

Offset para scroll a anclas (headers fijos).

html {
    scroll-padding-top: 80px;
    /* Deja espacio para header fijo */
}

4Scroll Snap

Crea puntos de "enganche" durante el scroll, ideal para carruseles y galerías.

Contenedor

.carousel {
    display: flex;
    overflow-x: auto;

    /* Tipo de snap */
    scroll-snap-type: x mandatory;
    /* x | y | both */
    /* mandatory: siempre para | proximity: cerca */

    /* Ocultar scrollbar */
    scrollbar-width: none;
}
.carousel::-webkit-scrollbar {
    display: none;
}

Ítems

.slide {
    flex: 0 0 100%;

    /* Punto de enganche */
    scroll-snap-align: start;
    /* start | center | end */

    /* Forzar parada */
    scroll-snap-stop: always;
    /* always: para en cada uno */
    /* normal: puede saltar */
}

Ejemplo: Carrusel de imágenes

.gallery {
    display: flex;
    gap: 1rem;
    overflow-x: auto;
    scroll-snap-type: x mandatory;
    scroll-padding: 1rem;
}

.gallery img {
    flex: 0 0 calc(100% - 2rem);
    scroll-snap-align: center;
    border-radius: 8px;
}

5Otras interacciones

touch-action

Controla gestos táctiles permitidos.

.element {
    touch-action: none;       /* Sin gestos */
    touch-action: pan-x;      /* Solo scroll horizontal */
    touch-action: pan-y;      /* Solo scroll vertical */
    touch-action: pinch-zoom; /* Solo zoom */
    touch-action: manipulation; /* Pan y zoom */
}

/* Ejemplo: mapa interactivo */
.map {
    touch-action: none;
}

pointer-events

Controla si un elemento responde a clics.

.disabled {
    pointer-events: none;
    /* No responde a clics ni hover */
    opacity: 0.5;
}

/* El clic pasa al elemento de abajo */
.overlay {
    pointer-events: none;
}

.overlay button {
    pointer-events: auto;  /* Restaurar */
}

cursor

.clickable { cursor: pointer; }
.loading { cursor: wait; }
.disabled { cursor: not-allowed; }
.text { cursor: text; }
.move { cursor: move; }
.resize { cursor: ew-resize; }
.custom { cursor: url(cursor.png), auto; }

user-select

Controla si el texto es seleccionable.

.button {
    user-select: none;  /* No seleccionable */
}

.code {
    user-select: all;   /* Selecciona todo al clic */
}
