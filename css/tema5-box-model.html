1El modelo de caja

Todo elemento HTML es una caja rectangular con 4 áreas:

margin
border
padding
content
  • Content: El contenido real (texto, imagen...)
  • Padding: Espacio interior entre contenido y borde
  • Border: El borde visible del elemento
  • Margin: Espacio exterior, separa de otros elementos

Elementos que no ocupan espacio

  • outline - Línea fuera del borde
  • box-shadow - Sombra del elemento
.box {
    /* Contenido */
    width: 200px;
    height: 100px;

    /* Padding: interior */
    padding: 20px;              /* Todos los lados */
    padding: 10px 20px;         /* Vertical | Horizontal */
    padding: 10px 20px 15px;    /* Top | Horizontal | Bottom */
    padding: 10px 20px 15px 5px; /* Top | Right | Bottom | Left */

    /* Border */
    border: 1px solid #333;
    border-width: 2px;
    border-style: solid | dashed | dotted | double | none;
    border-color: #333;

    /* Margin: exterior */
    margin: 20px;
    margin: 0 auto;             /* Centrar horizontalmente */
}

Tipos de caja según display

La propiedad display determina cómo se comporta la caja en el flujo del documento:

  • block - Ocupa todo el ancho disponible. Acepta width, height, margin y padding en todas las direcciones. (div, p, h1-h6, section...)
  • inline - Fluye con el texto. No acepta width ni height; margin y padding vertical no afectan al flujo. (span, a, strong, em...)
  • inline-block - Fluye en línea como inline, pero acepta el box model completo (width, height, margin, padding).

Ver Tema 6: Layout para una explicación ampliada de display.

2box-sizing

Controla cómo se calcula el tamaño total del elemento.

content-box (default)

width/height solo incluyen el contenido. Padding y border se suman.

.box {
    width: 200px;
    padding: 20px;
    border: 5px solid;
    /* Ancho total: 200 + 40 + 10 = 250px */
}

border-box (recomendado)

width/height incluyen contenido + padding + border.

.box {
    box-sizing: border-box;
    width: 200px;
    padding: 20px;
    border: 5px solid;
    /* Ancho total: 200px (siempre) */
}

/* Reset universal recomendado */
*, *::before, *::after {
    box-sizing: border-box;
}

3Dimensiones

Ancho y alto

.box {
    width: 300px;
    height: 200px;

    /* Límites */
    min-width: 100px;
    max-width: 800px;
    min-height: 50px;
    max-height: 500px;
}

Comportamiento por defecto

  • Block: width 100%, height auto (según contenido)
  • Inline: width y height según contenido (no se pueden cambiar)
  • Inline-block: width/height modificables, flujo inline

Margin auto

/* Centrar horizontalmente un bloque */
.centered {
    width: 80%;
    margin-inline: auto;
}

/* Margin negativo */
.overlap {
    margin-top: -20px;
}

4aspect-ratio

2020 - Define la proporción ancho/alto de un elemento. 

.video-container {
    width: 100%;
    aspect-ratio: 16 / 9;  /* Proporción de video HD */
}

.square {
    width: 200px;
    aspect-ratio: 1;       /* 1:1 = cuadrado */
}

.portrait {
    aspect-ratio: 3 / 4;   /* Tipo retrato */
}

/* Con iframes */
iframe {
    width: 100%;
    aspect-ratio: 16 / 9;
    height: auto;
}

Tip: Si defines width Y height explícitos, aspect-ratio se ignora. Define solo uno de los dos.

5object-fit y object-position

Controla cómo el contenido de <img>, <video> o <iframe> se ajusta a su contenedor.

object-fit

  • fill - Estira para llenar (puede distorsionar)
  • contain - Cabe entero, puede dejar espacio
  • cover - Cubre todo, puede recortar
  • none - Tamaño original
  • scale-down - El menor entre none y contain
.thumbnail {
    width: 150px;
    height: 150px;
    object-fit: cover;  /* Recorta para llenar */
}

.logo {
    width: 200px;
    height: 100px;
    object-fit: contain;  /* Cabe sin distorsionar */
}

object-position

.avatar {
    object-fit: cover;
    object-position: center top;  /* Enfoca en la cara */
}

.product {
    object-fit: contain;
    object-position: center center;
}

6Sizing keywords

Valores especiales para width/height que se adaptan al contenido.

min-content

El tamaño mínimo para que quepa el contenido (palabra más larga).

.tag {
    width: min-content;  /* Se ajusta a la palabra más larga */
}

max-content

El tamaño ideal del contenido sin wrapping.

.title {
    width: max-content;  /* Todo el texto en una línea */
}

fit-content

Entre min-content y max-content, con un límite opcional.

.card {
    width: fit-content;        /* Ajusta al contenido */
    max-width: 100%;           /* Pero no excede el padre */
}

.limited {
    width: fit-content(300px); /* Máximo 300px */
}

Uso común: Botones, etiquetas y elementos que deben ajustarse a su contenido pero no crecer indefinidamente.
