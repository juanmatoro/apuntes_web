1Fuentes y tipografía

Propiedades font-*

.text {
    font-family: 'Inter', Arial, sans-serif;
    font-size: 16px;
    font-weight: 400;      /* 100-900 o normal/bold */
    font-style: italic;    /* normal | italic | oblique */
    font-variant: small-caps;

    /* Atajo */
    font: italic 700 1.2rem/1.5 'Inter', sans-serif;
    /* style weight size/line-height family */
}

@font-face

@font-face {
    font-family: 'MiFuente';
    src: url('fuente.woff2') format('woff2'),
         url('fuente.woff') format('woff');
    font-weight: 400;
    font-style: normal;
    font-display: swap;  /* Evita FOIT */
}

/* font-display valores */
/* auto | block | swap | fallback | optional */

Fuentes variables

Un solo archivo con múltiples variaciones (peso, ancho, inclinación). 

@font-face {
    font-family: 'Inter';
    src: url('Inter-Variable.woff2') format('woff2-variations');
    font-weight: 100 900;
}

.element {
    font-variation-settings: 'wght' 450, 'slnt' -10;
}

2Propiedades de texto

Espaciado

.text {
    line-height: 1.5;        /* Altura de línea */
    letter-spacing: 0.05em;  /* Entre caracteres */
    word-spacing: 0.1em;     /* Entre palabras */
    text-indent: 2em;        /* Sangría primera línea */
}

Alineación

.text {
    text-align: left | center | right | justify;

    /* Propiedades lógicas */
    text-align: start;       /* Según dirección */
    text-align: end;

    vertical-align: baseline | top | middle | bottom;
}

Decoración

.text {
    text-decoration: underline;
    text-decoration: line-through wavy red;

    /* Propiedades individuales */
    text-decoration-line: underline | overline | line-through;
    text-decoration-style: solid | dashed | dotted | wavy;
    text-decoration-color: red;
    text-decoration-thickness: 2px;
    text-underline-offset: 3px;
}

Transformación

.text {
    text-transform: uppercase;
    text-transform: lowercase;
    text-transform: capitalize;
    text-transform: none;
}

/* Sombra de texto */
.text {
    text-shadow: 2px 2px 4px rgba(0,0,0,0.3);
    /* offset-x offset-y blur color */
}

Control de espacio en blanco y palabras

.text {
    white-space: normal | nowrap | pre | pre-wrap | pre-line;
    word-break: normal | break-all | keep-all;
    overflow-wrap: normal | break-word | anywhere;
    hyphens: none | manual | auto;
}

3Text-wrap

2023 - Controla cómo se distribuye el texto en múltiples líneas. 

.title {
    text-wrap: balance;
    /* Distribuye texto uniformemente
       (evita líneas huérfanas cortas) */
}

.paragraph {
    text-wrap: pretty;  /* 2024 */
    /* Optimiza saltos de línea
       para mejor lectura */
}

.code {
    text-wrap: nowrap;
    /* Sin saltos de línea */
}

line-clamp

Limita el texto a un número de líneas.

.excerpt {
    display: -webkit-box;
    -webkit-box-orient: vertical;
    -webkit-line-clamp: 3;
    overflow: hidden;
}

4Listas

list-style

ul {
    list-style-type: disc | circle | square | none;
    list-style-position: inside | outside;
    list-style-image: url('bullet.svg');

    /* Atajo */
    list-style: square inside;
}

ol {
    list-style-type: decimal | lower-alpha | upper-roman;
}

::marker

Estiliza el marcador de listas.

li::marker {
    color: tomato;
    font-size: 1.2em;
    font-weight: bold;
}

/* Contenido personalizado */
li::marker {
    content: "→ ";
}

/* En summary/details */
summary::marker {
    content: "▸ ";
}
details[open] summary::marker {
    content: "▾ ";
}

5Modos de escritura

Controla la dirección del texto (horizontal, vertical, RTL).

writing-mode

.horizontal {
    writing-mode: horizontal-tb;  /* Default */
}

.vertical {
    writing-mode: vertical-rl;    /* Vertical, derecha a izquierda */
    writing-mode: vertical-lr;    /* Vertical, izquierda a derecha */
}

/* Uso: texto lateral en diseño */
.sidebar-title {
    writing-mode: vertical-lr;
    transform: rotate(180deg);
}

direction y text-orientation

.rtl {
    direction: rtl;   /* Derecha a izquierda (árabe, hebreo) */
    direction: ltr;   /* Izquierda a derecha (default) */
}

.vertical {
    writing-mode: vertical-rl;
    text-orientation: mixed;    /* Default */
    text-orientation: upright;  /* Caracteres verticales */
    text-orientation: sideways; /* Caracteres girados */
}

Nota: Los modos de escritura afectan a las propiedades lógicas. Por eso inline-size es mejor que width para sitios multilingües.
