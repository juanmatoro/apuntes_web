1Custom Properties

Dinamismo: Se resuelven en el navegador en tiempo real. Pueden cambiarse con JS o dentro de Media Queries sin duplicar código.

Definición

Fallbacks

2calc()

⚠️ Espacios obligatorios: calc(100% - 20px). 
.content { width: calc(100% - 60px); }

3Comparación

min(100%, 500px): No pases de 500px.
clamp(min, ideal, max): Fluidez total.

4Escalas y Fluidez

Escala Modular

Usa un ratio (ej. 1.25) para que cada paso sea proporcional.

--step-1: calc(var(--step-0) * 1.25);

Clamp Líquido

La base cambia con el viewport sin Media Queries.

--fs: clamp(1rem, 5vw, 1.2rem);
