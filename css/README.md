## Características destacadas por año

### 2024
- `@scope` - Alcance de selectores
- Sintaxis de color relativa (`from`)
- Container style queries
- Scroll-linked animations (`scroll()`, `view()`)
- `light-dark()`
- Funciones matemáticas: `round()`, `mod()`, `abs()`, `sign()`

### 2023
- Anidamiento nativo de selectores
- `:has()` - Selector padre
- Container size queries
- `text-wrap: balance`
- View Transitions API
- `image-set()`

### 2022
- `@layer` - Capas de cascada
- `:where()` - Sin especificidad
- `color-mix()`
- Unidades de viewport variables (`svh`, `lvh`, `dvh`)
- Media queries con rangos
- `scrollbar-gutter`
- Funciones trigonométricas

### 2021
- `@property` (Houdini)
- `accent-color`
- `gap` en Flexbox
- `::marker`
- Fuentes variables

### 2020
- `aspect-ratio`
- `position: sticky`
- `content-visibility`
- `prefers-color-scheme`
- `prefers-reduced-motion`
