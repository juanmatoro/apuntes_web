1Media Queries

Aplican estilos según características del dispositivo o preferencias del usuario.

Sintaxis

/* Tradicional */
@media (max-width: 768px) {
    .sidebar { display: none; }
}

/* Moderna (2022) - con operadores */
@media (width >= 768px) {
    .container { max-width: 720px; }
}

/* Rango */
@media (400px <= width <= 800px) {
    .card { padding: 1rem; }
}

/* Combinaciones */
@media (min-width: 768px) and (orientation: landscape) {
    /* ... */
}

@media (max-width: 480px), (orientation: portrait) {
    /* O una u otra */
}

Breakpoints comunes

/* Mobile first */
/* Base: móvil */

@media (min-width: 640px) {
    /* sm: tablet pequeña */
}

@media (min-width: 768px) {
    /* md: tablet */
}

@media (min-width: 1024px) {
    /* lg: desktop */
}

@media (min-width: 1280px) {
    /* xl: desktop grande */
}

@media (min-width: 1536px) {
    /* 2xl: pantallas muy grandes */
}

Otras características

@media (orientation: portrait) { }    /* Vertical */
@media (orientation: landscape) { }   /* Horizontal */
@media (hover: hover) { }             /* Dispositivo con hover */
@media (pointer: coarse) { }          /* Touch */
@media (pointer: fine) { }            /* Mouse/stylus */
@media print { }                      /* Para impresión */
@media (display-mode: standalone) { } /* PWA instalada */

2Container Queries

2023 - Estilos basados en el tamaño del contenedor, no del viewport.

Container size queries

/* Definir contenedor */
.card-container {
    container-type: inline-size;
    container-name: card;
    /* O atajo: */
    container: card / inline-size;
}

/* Consultar tamaño del contenedor */
@container card (min-width: 400px) {
    .card {
        display: flex;
        gap: 1rem;
    }
}

@container (width > 300px) {
    /* Sin nombre: contenedor más cercano */
}

Container style queries (2024)

/* Consultar custom properties del contenedor */
.card-container {
    --theme: dark;
}

@container style(--theme: dark) {
    .card {
        background: #1a1a1a;
        color: white;
    }
}

/* Combinado con tamaño */
@container card (width > 400px) and style(--featured: true) {
    .card {
        border: 2px solid gold;
    }
}

Unidades de contenedor

.card-title {
    font-size: clamp(1rem, 5cqi, 2rem);
    /* cqi: 1% del inline-size del contenedor */
    /* cqw, cqh, cqmin, cqmax también disponibles */
}

3Accesibilidad (prefers-*)

Media queries que respetan las preferencias del usuario.

prefers-color-scheme

/* Tema oscuro automático */
@media (prefers-color-scheme: dark) {
    :root {
        --bg: #1a1a1a;
        --text: #f0f0f0;
    }
}

/* O usar color-scheme */
:root {
    color-scheme: light dark;
}

light-dark() (2024)

.element {
    background: light-dark(white, #1a1a1a);
    color: light-dark(#333, #f0f0f0);
}

prefers-reduced-motion

@media (prefers-reduced-motion: reduce) {
    * {
        animation-duration: 0.01ms !important;
        transition-duration: 0.01ms !important;
    }
}

prefers-contrast

@media (prefers-contrast: more) {
    .button {
        border: 2px solid currentColor;
    }
}

forced-colors

@media (forced-colors: active) {
    .button {
        border: 2px solid ButtonText;
    }
}

prefers-reduced-data

@media (prefers-reduced-data: reduce) {
    .hero {
        background-image: none;
    }
}

4@supports (Feature queries)

Aplica estilos solo si el navegador soporta una característica.

/* Si soporta grid */
@supports (display: grid) {
    .container {
        display: grid;
    }
}

/* Si NO soporta */
@supports not (display: grid) {
    .container {
        display: flex;
    }
}

/* Combinaciones */
@supports (display: grid) and (gap: 1rem) {
    /* ... */
}

/* Verificar selector */
@supports selector(:has(*)) {
    /* Usar :has() */
}

/* Verificar @property */
@supports (background: paint(something)) {
    /* Houdini Paint API disponible */
}

5Rendimiento

content-visibility

Omite renderizado de elementos fuera del viewport.

.section {
    content-visibility: auto;
    contain-intrinsic-size: 0 500px;  /* Tamaño estimado */
}

contain

Limita el ámbito de cálculos del navegador.

.widget {
    contain: layout;    /* Aislado del resto */
    contain: paint;     /* No pinta fuera */
    contain: size;      /* Tamaño independiente */
    contain: style;     /* Contadores aislados */
    contain: strict;    /* layout + paint + size */
    contain: content;   /* layout + paint */
}

will-change

Avisa al navegador sobre cambios futuros.

.animated {
    will-change: transform, opacity;
}
/* Usar con moderación, consume memoria */

Buenas prácticas

  • Animar solo transform y opacity (GPU)
  • Evitar animaciones de width/height
  • Usar font-display: swap para fuentes
  • Cargar CSS crítico inline
