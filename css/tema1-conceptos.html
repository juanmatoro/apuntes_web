1Selectores

Partes de una regla CSS

selector {
    propiedad: valor;
}

Selectores simples

  • * - Selector universal (todos los elementos)
  • elemento - Selector de tipo (p, div, h1...)
  • .clase - Selector de clase
  • #id - Selector de ID
  • [atributo] - Selector de atributo
  • [attr="valor"] - Atributo con valor exacto
  • [attr^="inicio"] - Atributo que empieza por...
  • [attr$="fin"] - Atributo que termina en...
  • [attr*="contiene"] - Atributo que contiene...

Selectores combinados (combinadores)

  • A B - Descendiente (B dentro de A, cualquier nivel)
  • A > B - Hijo directo (B hijo inmediato de A)
  • A + B - Hermano adyacente (B inmediatamente después de A)
  • A ~ B - Hermano general (B después de A, mismo nivel)
/* Ejemplos de combinadores */
article p { }       /* p dentro de article */
ul > li { }         /* li hijo directo de ul */
h2 + p { }          /* p justo después de h2 */
h2 ~ p { }          /* todos los p hermanos de h2 */

Selectores compuestos

  • A, B - Agrupación (A o B)
  • A.clase - Elemento A con clase
  • A#id - Elemento A con ID

2Anidamiento de selectores

2023 - Permite escribir CSS anidado de forma nativa, similar a SASS/LESS.

  • Reduce la repetición de selectores
  • Mejora la legibilidad y organización
  • El símbolo & referencia al selector padre
/* Anidamiento nativo en CSS */
.card {
    padding: 1rem;

    .title {
        font-size: 1.5rem;
    }

    &:hover {
        box-shadow: 0 4px 8px rgba(0,0,0,0.1);
    }

    &.featured {
        border: 2px solid gold;
    }
}

3@scope

2024 - Nueva regla CSS que delimita el alcance de los selectores.

  • @scope (inicio) - Define dónde empiezan a aplicarse los estilos
  • @scope (inicio) to (fin) - Define inicio y fin del alcance
  • Evita fugas de estilos a elementos no deseados
/* @scope básico */
@scope (.card) {
    p { color: gray; }
    .title { font-weight: bold; }
}

/* @scope con límite */
@scope (.card) to (.card-footer) {
    /* Estilos solo entre .card y .card-footer */
    p { margin: 0.5rem 0; }
}

4La cascada

El algoritmo que determina qué estilos se aplican cuando hay conflictos. Orden de prioridad:

  1. Origen e importancia
    • Estilos del agente de usuario (navegador)
    • Estilos del usuario
    • Estilos del autor (tu CSS)
    • !important invierte el orden
  2. Especificidad - Peso del selector (ver sección siguiente)
  3. Orden de aparición - El último declarado gana

@layer - Capas de cascada

2022 - Permite organizar estilos en capas con orden de prioridad controlado. 

/* Declarar orden de capas */
@layer reset, base, components, utilities;

/* Definir estilos en cada capa */
@layer reset {
    * { margin: 0; padding: 0; }
}

@layer base {
    body { font-family: sans-serif; }
}

@layer components {
    .btn { padding: 0.5rem 1rem; }
}

@layer utilities {
    .mt-4 { margin-top: 1rem; }
}

Las capas declaradas primero tienen menor prioridad. Los estilos sin @layer tienen la mayor prioridad.

5Especificidad

Sistema de puntuación que determina qué selector tiene prioridad.

Selector Puntos
*, combinadores0
Elemento, pseudo-elemento0-0-1
Clase, atributo, pseudo-clase0-1-0
ID1-0-0
style="" (inline)1-0-0-0
!importantGana siempre*
 
/* Ejemplos de especificidad */
p { }                  /* 0-0-1 */
.card { }              /* 0-1-0 */
#header { }            /* 1-0-0 */
p.intro { }            /* 0-1-1 */
#nav .link:hover { }   /* 1-2-0 */
div#main p.text { }    /* 1-1-2 */

6Herencia

Algunas propiedades CSS se heredan automáticamente de padres a hijos.

Propiedades heredables

  • Texto: color, font-*, letter-spacing, line-height, text-align, text-indent, text-transform
  • Listas: list-style-*
  • Otros: visibility, cursor

Propiedades NO heredables

  • Box model: margin, padding, border, width, height
  • Posicionamiento: position, top, left, z-index
  • Fondo: background-*

Controlar la herencia

.elemento {
    color: inherit;    /* Fuerza herencia del padre */
    margin: initial;   /* Valor por defecto del navegador */
    padding: unset;    /* inherit si hereda, initial si no */
    all: revert;       /* Revierte a estilos del navegador */
}
