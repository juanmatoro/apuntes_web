1La propiedad display

Valores básicos

  • block - Ocupa todo el ancho, nueva línea
  • inline - Fluye con el texto, sin dimensiones
  • inline-block - Inline con dimensiones
  • none - Oculta completamente

Valores de layout

  • flex - Contenedor Flexbox
  • inline-flex - Flexbox inline
  • grid - Contenedor Grid
  • inline-grid - Grid inline

2Flexbox

Sistema de layout unidimensional (fila O columna).

Contenedor (padre)

.container {
    display: flex;

    /* Dirección */
    flex-direction: row;        /* default */
    flex-direction: column;
    flex-direction: row-reverse;

    /* Wrap */
    flex-wrap: nowrap;          /* default */
    flex-wrap: wrap;

    /* Atajo */
    flex-flow: row wrap;

    /* Alineación eje principal */
    justify-content: flex-start;
    justify-content: center;
    justify-content: flex-end;
    justify-content: space-between;
    justify-content: space-around;
    justify-content: space-evenly;

    /* Alineación eje cruzado */
    align-items: stretch;       /* default */
    align-items: flex-start;
    align-items: center;
    align-items: flex-end;
    align-items: baseline;

    /* Múltiples líneas */
    align-content: flex-start;
}

Ítems (hijos)

.item {
    /* Orden visual */
    order: 0;                   /* default */

    /* Crecimiento */
    flex-grow: 0;               /* No crece */
    flex-grow: 1;               /* Crece proporcionalmente */

    /* Encogimiento */
    flex-shrink: 1;             /* default, puede encogerse */

    /* Base del tamaño */
    flex-basis: auto;           /* Según contenido */
    flex-basis: 200px;          /* Tamaño inicial */

    /* Atajo */
    flex: 1;                    /* grow:1 shrink:1 basis:0 */
    flex: 0 0 200px;            /* No crece, no encoge, 200px */

    /* Alineación individual */
    align-self: center;
}

Centrar elemento

.center {
    display: flex;
    justify-content: center;
    align-items: center;
}

3CSS Grid

Sistema de layout bidimensional (filas Y columnas).

Contenedor

.grid {
    display: grid;

    /* Columnas */
    grid-template-columns: 200px 200px 200px;
    grid-template-columns: 1fr 2fr 1fr;
    grid-template-columns: repeat(3, 1fr);
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));

    /* Filas */
    grid-template-rows: 100px auto 100px;

    /* Filas implícitas (auto-generadas) */
    grid-auto-rows: minmax(100px, auto);

    /* Alineación */
    justify-items: start | center | end | stretch;
    align-items: start | center | end | stretch;

    /* Alineación del grid completo */
    justify-content: center;
    align-content: center;
}

Ítems

.item {
    /* Posición por líneas */
    grid-column: 1 / 3;         /* Línea 1 a 3 */
    grid-column: span 2;        /* Ocupa 2 columnas */
    grid-row: 1 / 2;

    /* Atajos */
    grid-area: 1 / 1 / 3 / 3;   /* row-start / col-start / row-end / col-end */

    /* Alineación individual */
    justify-self: center;
    align-self: center;
}

Grid Areas (nombradas)

.layout {
    grid-template-areas:
        "header header"
        "sidebar main"
        "footer footer";
}
.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

Funciones útiles

/* minmax() - Rango de tamaño */
grid-template-columns: minmax(100px, 1fr) 2fr;

/* repeat() - Repetir patrón */
grid-template-columns: repeat(4, 1fr);

/* auto-fit vs auto-fill */
repeat(auto-fit, minmax(200px, 1fr))   /* Estira para llenar */
repeat(auto-fill, minmax(200px, 1fr))  /* Mantiene huecos */

/* La unidad fr - fracción del espacio disponible */
grid-template-columns: 1fr 2fr 1fr;    /* 25% 50% 25% */

4Subgrid

Permite que un elemento hijo herede las líneas del grid padre. 

.grid {
    display: grid;
    grid-template-columns: repeat(4, 1fr);
    grid-template-rows: auto auto auto;
}

.card {
    grid-column: span 2;
    grid-row: span 3;

    /* Hereda las líneas del padre */
    display: grid;
    grid-template-columns: subgrid;
    grid-template-rows: subgrid;
}

/* Los hijos de .card se alinean
   con las líneas del grid principal */
.card-title { grid-row: 1; }
.card-body { grid-row: 2; }
.card-footer { grid-row: 3; }

Uso: Alinear elementos de tarjetas (título, contenido, footer) entre sí, incluso con contenido de diferentes longitudes.

5Gap

2021 - Espaciado entre elementos en Flex y Grid. 

.container {
    display: flex; /* o grid */

    /* Gap uniforme */
    gap: 20px;

    /* Filas y columnas diferentes */
    gap: 10px 20px;      /* row-gap | column-gap */

    /* Propiedades individuales */
    row-gap: 10px;
    column-gap: 20px;
}

Ventajas sobre margin

  • Solo aplica entre elementos (no en bordes)
  • No necesita selectores :first-child/:last-child
  • No hay colapso de márgenes
  • Funciona con wrap en Flexbox

6Multi-column layout

Divide el contenido en múltiples columnas, como en periódicos.

Propiedades principales

.article {
    /* Número de columnas */
    column-count: 3;

    /* O ancho de columna (auto-calcula cantidad) */
    column-width: 200px;

    /* Atajo: auto para uno, valor para otro */
    columns: 3;
    columns: 200px;
    columns: 3 200px;

    /* Espacio entre columnas */
    column-gap: 40px;

    /* Línea separadora */
    column-rule: 1px solid #ccc;
}

Control de contenido

/* Evitar que un elemento se corte entre columnas */
.article h2 {
    break-after: avoid;
}

.article figure {
    break-inside: avoid;
}

/* Elemento que ocupa todas las columnas */
.article .featured {
    column-span: all;
}

Uso: Artículos largos, galerías tipo masonry, listas en columnas.
