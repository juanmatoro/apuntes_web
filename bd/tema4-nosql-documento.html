Modelo Consultas Trade-offs

1Documento y coleccion

En bases documento, cada fila es un JSON flexible. Es util para datos cambiantes y jerarquias naturales.

2Operaciones frecuentes

db.usuarios.find({ activo: true }, { email: 1, rol: 1 });
db.usuarios.updateOne({ _id: id }, { $set: { ultimoLogin: new Date() } });

3Consistencia y escalado

Prioriza disponibilidad y particionado horizontal, asumiendo en algunos casos consistencia eventual.
