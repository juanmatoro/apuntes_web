DDL y DML Consultas Joins

1DDL y DML

CREATE TABLE usuario (
  id INT PRIMARY KEY,
  email VARCHAR(120) UNIQUE
);

INSERT INTO usuario (id, email) VALUES (1, 'ana@mail.com');

2SELECT y filtros

SELECT id, email
FROM usuario
WHERE email LIKE '%@mail.com'
ORDER BY id DESC;

Usa INNER JOIN para coincidencias y LEFT JOIN cuando necesites conservar filas de la tabla izquierda aunque no haya match.
