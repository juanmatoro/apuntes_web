Schema Migraciones Cliente

1Definir el schema

model Usuario {
  id    Int    @id @default(autoincrement())
  email String @unique
  nombre String
}

2Migraciones

npx prisma migrate dev --name init
npx prisma generate

Cada cambio estructural del schema debe versionarse como migracion.
