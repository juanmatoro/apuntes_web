Expresiones Tipos DOM

1Statement vs Expression

Código

const total = 2 + 2;
console.log(total);

Idea clave

2 + 2 es una expresión (produce valor). La línea completa con const es un statement (ejecuta acción).

2Tipos y coerción

Código

console.log(typeof 42);       // "number"
console.log(typeof "42");     // "string"
console.log("1" + 1);         // "11" (coerción)

Idea clave

JavaScript es de tipado dinámico y aplica coerción en ciertas operaciones. Conviene conocer cuándo convierte tipos automáticamente.

3Primer contacto con DOM

Código

const titulo = document.querySelector("h1");
if (titulo) {
  titulo.textContent = "JavaScript en acción";
}

Idea clave

El DOM representa el HTML en memoria. Con JavaScript puedes leerlo y modificarlo en tiempo de ejecución.
