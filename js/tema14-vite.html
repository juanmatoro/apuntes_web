1Qué es Vite y por qué se usa

Vite es una herramienta de desarrollo para frontend que optimiza dos fases clave: servidor de desarrollo y build de producción.

  • En desarrollo usa módulos ESM nativos y arranca muy rápido.
  • En producción genera assets optimizados con bundling.
  • Es una base habitual para proyectos Vanilla JS, Vue, React y otros frameworks.

2Crear y arrancar un proyecto

Flujo mínimo de trabajo con npm:

npm create vite@latest mi-app
cd mi-app
npm install
npm run dev

El comando npm run dev inicia el servidor local con recarga inmediata cuando guardas cambios.

3Estructura base del proyecto 

mi-app/
  index.html
  package.json
  public/
  src/
    main.js
    style.css
  • index.html es punto de entrada en desarrollo.
  • src/main.js contiene el arranque de la app.
  • public/ guarda estáticos que se copian tal cual.

4Scripts clave: dev, build y preview 

{
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "preview": "vite preview"
  }
}
  • dev: servidor de desarrollo.
  • build: genera versión optimizada en dist/.
  • preview: sirve localmente el build para validarlo antes de desplegar.

5Configuración básica con vite.config.js 

import { defineConfig } from 'vite';

export default defineConfig({
  server: {
    port: 5173,
    open: true
  },
  build: {
    sourcemap: true
  }
});

Esta configuración controla el servidor local y opciones de compilación de salida.
