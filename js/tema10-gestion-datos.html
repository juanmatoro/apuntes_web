1JSON como formato de intercambio

JSON es un formato de texto para representar datos estructurados. Se usa para enviar y guardar información.

Serializar

const user = { id: 1, nombre: 'Ana' };
const json = JSON.stringify(user);

Parsear

const raw = '{"id":1,"nombre":"Ana"}';
const user = JSON.parse(raw);

2Obtención de datos desde formularios

Puedes leer controles del formulario por FormData o por acceso directo a inputs.

form.addEventListener('submit', (event) => {
  event.preventDefault();
  const data = new FormData(form);
  const payload = Object.fromEntries(data.entries());
  console.log(payload);
});

3Validación de datos

Valida tanto en UI (feedback rápido) como en capa de negocio (reglas reales).

function validarRegistro({ email, password }) {
  const errores = [];
  if (!email.includes('@')) errores.push('Email inválido');
  if (password.length < 8) errores.push('Password demasiado corta');
  return errores;
}

4Persistencia local (localStorage)

Para datos pequeños del cliente, localStorage es práctico si serializas a JSON.

const KEY = 'tasks';

function saveTasks(tasks) {
  localStorage.setItem(KEY, JSON.stringify(tasks));
}

function loadTasks() {
  const raw = localStorage.getItem(KEY);
  return raw ? JSON.parse(raw) : [];
}
