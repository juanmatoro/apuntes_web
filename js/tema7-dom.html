DOM Nodos Acceso Recorrido Manipulación

1DOM (Document Object Model)

El DOM es la representación en memoria del HTML como un árbol de nodos. JavaScript lo usa como interfaz para leer y modificar la página.

  • El navegador parsea HTML y CSS y construye el árbol DOM.
  • Los cambios en el DOM se reflejan en pantalla.

2Tipos de nodos

Un documento está formado por distintos tipos de nodos. Los más comunes son:

Principales

  • Elemento: ELEMENT_NODE
  • Texto: TEXT_NODE
  • Documento: DOCUMENT_NODE

Otros

  • Atributo: ATTRIBUTE_NODE
  • Comentario: COMMENT_NODE
  • Fragmento: DOCUMENT_FRAGMENT_NODE

3Acceso a nodos

Puedes acceder al DOM con selectores CSS o métodos clásicos.

Query

const el = document.querySelector('.card');
const all = document.querySelectorAll('li');

Get

document.getElementById('app');
document.getElementsByClassName('item');
document.getElementsByTagName('li');

4Recorrido del árbol DOM

Cada nodo tiene referencias para moverte por el árbol.

Hijos

el.children;
el.firstElementChild;
el.lastElementChild;

Padres y hermanos

el.parentElement;
el.nextElementSibling;
el.previousElementSibling;

5Creación y modificación

Puedes crear nodos y modificar contenido, atributos y clases.

Crear y añadir

const li = document.createElement('li');
li.textContent = 'Nuevo';
list.appendChild(li);

Atributos y clases

el.setAttribute('data-id', '7');
el.classList.add('active');
el.classList.toggle('hidden');

innerHTML / textContent

title.textContent = 'Hola';
container.innerHTML = '<strong>OK</strong>';

insertAdjacentHTML

card.insertAdjacentHTML('beforeend', '
Nuevo
');
