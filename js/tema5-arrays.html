1Arrays básicos

Un array es una colección ordenada de valores. Se accede por índice (desde 0). ((En JavaScript, los arrays pueden contener cualquier tipo de dato, incluso otros arrays o funciones. y su notacion es con corchetes [] para crear y acceder a elementos.))

Creación y acceso 

const colores = ['rojo', 'verde', 'azul'];

colores[0]; // 'rojo'
colores.length; // 3
colores.at(-1); // 'azul'

Array con tipos mixtos 

const data = [1, 'hola', true, { x: 1 }];

Array.isArray(data); // true

2Iteración y tipos de bucle

Iterar es repetir una acción sobre los elementos de un array. Cada bucle encaja mejor en un tipo de problema.

Bucle Cuándo usarlo Uso práctico
for Necesitas índice o control fino. Recorrido por posiciones, saltos personalizados.
while No conoces cuántas vueltas habrá. Reintentos hasta cumplir condición.
do...while El bloque debe ejecutarse al menos una vez. Lectura/validación inicial obligatoria.
for...of Quieres valores directos del array. Procesar elementos sin manejar índices.
forEach Quieres iterar de forma declarativa con callback. Efectos por elemento (logs, pintar UI, acumuladores externos).

for y for...of 

const items = ['a', 'b', 'c'];

for (let i = 0; i < items.length; i++) {
  console.log(i, items[i]); // con índice
}

for (const item of items) {
  console.log(item); // solo valor
}

while, do...while y forEach 

let intentos = 0;
let ok = false;
while (!ok && intentos < 3) {
  intentos++;
  ok = Math.random() > 0.5;
}

let n = 0;
do {
  n++;
} while (n < 1);

const tareas = ['leer', 'practicar', 'repasar'];
tareas.forEach((t, i) => {
  console.log(i, t);
});

Regla rápida

  • Array con índice: for.
  • Array solo por valor: for...of.
  • Efectos por elemento con callback: forEach.
  • Iteraciones indeterminadas: while.
  • Al menos una ejecución: do...while.

3Métodos de arrays más usados

Metodos

Métodos HOF (reciben callback)

Método ¿Modifica el original? ¿Qué devuelve? Propósito principal
map No Nuevo array Transformar datos
filter No Nuevo array Descartar elementos
reduce No Un solo valor Acumular / Agrupar
forEach No (pero invita a hacerlo) undefined Ejecutar efectos (ej. un console.log)
find No Un elemento, el primero que cumpla la condición Buscar un objeto específico
findIndex No Índice del primer elemento que cumpla, o -1 si no cumple ninguno Encontrar la posición de un elemento
flatMap No Nuevo array aplanado 1 nivel Mapear y aplanar en una sola operación

Métodos no HOF

Método ¿Modifica el original? ¿Qué devuelve? Propósito principal
slice No Nuevo array Cortar una parte del array
flat No Nuevo array aplanado Aplanar arrays anidados
splice Sí Elementos eliminados Insertar o borrar elementos

Transformar, filtrar y buscar 

const precios = [20, 35, 50, 12];

const conIva = precios.map(p => p * 1.21);
const caros = precios.filter(p => p > 30);
const primeroCaro = precios.find(p => p > 30);

Acumular, cortar y editar 

const nums = [10, 20, 30, 40];
const total = nums.reduce((acc, n) => acc + n, 0); // 100
const tramo = nums.slice(1, 3); // [20, 30]

const copia = [...nums];
const quitados = copia.splice(1, 2, 25, 35);
// copia => [10, 25, 35, 40]
// quitados => [20, 30]

Mutables (modifican el original) 

const nums = [3, 1, 2];
nums.push(4);     // [3,1,2,4]
nums.pop();       // [3,1,2]
nums.sort();      // [1,2,3]

Inmutables (devuelven nuevo array) 

const nums = [3, 1, 2];
const ordenado = nums.toSorted(); // [1,2,3]
const pares = nums.filter(n => n % 2 === 0);
const dobles = nums.map(n => n * 2);

4Patrones de uso en proyectos

En la web, los arrays se usan para renderizar listas, filtrar datos y gestionar estados.

Render de listas

const tasks = ['A', 'B', 'C'];
const html = tasks.map(t => `<li>${t}</li>`).join('');
list.innerHTML = html;

Ejemplo: Tres en raya 

const board = ['X', '', 'O', '', 'X', '', '', '', 'O'];
const winner = checkWinner(board);

function checkWinner(cells) {
  const lines = [
    [0,1,2],[3,4,5],[6,7,8],
    [0,3,6],[1,4,7],[2,5,8],
    [0,4,8],[2,4,6]
  ];
  return lines.find(([a, b, c]) =>
    cells[a] && cells[a] === cells[b] && cells[a] === cells[c]
  );
}
