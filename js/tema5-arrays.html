1Arrays básicos

Un array es una colección ordenada de valores. Se accede por índice (desde 0).

Creación y acceso

const colores = ['rojo', 'verde', 'azul'];

colores[0]; // 'rojo'
colores.length; // 3
colores.at(-1); // 'azul'

Array con tipos mixtos

const data = [1, 'hola', true, { x: 1 }];

Array.isArray(data); // true

2Métodos mutables e inmutables

Los métodos mutables modifican el array original; los inmutables devuelven uno nuevo.

Mutables

const nums = [3, 1, 2];
nums.push(4);     // [3,1,2,4]
nums.pop();       // [3,1,2]
nums.sort();      // [1,2,3]

Inmutables

const nums = [3, 1, 2];
const ordenado = nums.toSorted(); // [1,2,3]
const pares = nums.filter(n => n % 2 === 0);
const dobles = nums.map(n => n * 2);

3Iteración y búsqueda

Para recorrer arrays usa for, for...of o métodos como forEach.

for...of

const items = ['a', 'b', 'c'];
for (const item of items) {
  console.log(item);
}

find / some / every

const nums = [1, 2, 3, 4];
nums.find(n => n > 2);   // 3
nums.some(n => n > 3);   // true
nums.every(n => n > 0);  // true

4Patrones de uso en proyectos

En la web, los arrays se usan para renderizar listas, filtrar datos y gestionar estados.

Render de listas

const tasks = ['A', 'B', 'C'];
const html = tasks.map(t => `<li>${t}</li>`).join('');
list.innerHTML = html;

Ejemplo: Tres en raya

const board = ['X', '', 'O', '', 'X', '', '', '', 'O'];
const winner = checkWinner(board);

function checkWinner(cells) {
  const lines = [
    [0,1,2],[3,4,5],[6,7,8],
    [0,3,6],[1,4,7],[2,5,8],
    [0,4,8],[2,4,6]
  ];
  return lines.find(([a, b, c]) =>
    cells[a] && cells[a] === cells[b] && cells[a] === cells[c]
  );
}
