2Funciones anónimas y arrow

Una función anónima no tiene nombre. Las arrow permiten escribir versiones más compactas.

const saludar = function(nombre) {
  return `Hola ${nombre}`;
};

const saludarCorto = nombre => `Hola ${nombre}`; // mínima expresión (estilo lambda)

3Funciones callback

Un callback es una función que se pasa como argumento para ser ejecutada por otra función.

function ejecutar(a, b, callback) {
  return callback(a, b);
}

const resta = ejecutar(8, 3, function(x, y) {
  return x - y;
});

const producto = ejecutar(8, 3, (x, y) => x * y);

4Funciones HOF

Una HOF recibe o devuelve funciones. Cuando recibe una función, esa función es el callback.

const nums = [1, 2, 3, 4, 5];

const pares = nums.filter(n => n % 2 === 0); // filter es HOF, callback: n => n % 2 === 0

function crearMultiplicador(factor) {
  return n => n * factor; // HOF que devuelve función
}

const por2 = crearMultiplicador(2);
por2(6); // 12

5Errores comunes 

// Error 1: confundir callback con HOF
const cb = n => n * 2; // callback
// cb no es HOF por sí sola

// Error 2: usar arrow como método cuando necesitas this
const usuario = {
  nombre: 'Ana',
  saludar: () => this.nombre // undefined en muchos contextos
};

// Correcto:
const usuario2 = {
  nombre: 'Ana',
  saludar() {
    return this.nombre;
  }
};
