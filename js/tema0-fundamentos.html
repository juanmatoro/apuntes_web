1Qué es JavaScript

JavaScript es el lenguaje que aporta la capa interactiva de una web. Según web.dev/learn/javascript, complementa la estructura (HTML) y la presentación (CSS), permitiendo cambiar contenido y comportamiento en respuesta a eventos y lógica interna.

HTML

Estructura del documento.

CSS

Presentación y estilos visuales.

JavaScript

Interactividad, lógica y comportamiento.

2ECMAScript y JavaScript

  • ECMAScript (ES) es el estándar (ECMA-262).
  • JavaScript es la implementación más usada de ese estándar.
  • El estándar evoluciona por ediciones (ES2015, ES2016, ...).

Regla práctica: cuando hablamos de sintaxis y reglas del lenguaje, hablamos de ECMAScript; cuando hablamos de su uso real en navegador o Node.js, hablamos de JavaScript en un entorno concreto.

3JavaScript en la Web: mapa rápido

En navegador, JavaScript se ejecuta junto al motor de renderizado y puede leer/modificar el DOM.

HTML + CSS + JS
  -> Parsing (análisis)
  -> DOM + CSSOM
  -> Render Tree
  -> Layout
  -> Paint

Este flujo se desarrolla con más detalle en el Tema 1 (arquitectura del navegador).

4APIs de navegador (Web APIs)

Según MDN, las Web APIs son interfaces que expone el navegador para que JavaScript pueda interactuar con capacidades del entorno (red, almacenamiento, interfaz, sensores, historial, etc.). No forman parte del núcleo ECMAScript: pertenecen al runtime del navegador.

API Para qué sirve Ejemplo común
DOM API Leer y modificar el documento HTML. document.querySelector()
Fetch API Hacer peticiones HTTP asíncronas. fetch('/api')
Web Storage API Asignar y leer pares clave-valor en el navegador. localStorage.setItem()
Geolocation API Obtener ubicación del usuario (con permiso). navigator.geolocation.getCurrentPosition()
History API Gestionar historial y URL sin recargar. history.pushState()

Ejemplo: Fetch API

async function cargarUsuarios() {
  const res = await fetch('/api/users');
  if (!res.ok) throw new Error('Error HTTP');
  return res.json();
}

Ejemplo: Web Storage API

localStorage.setItem('temaActual', 'fundamentos');
const tema = localStorage.getItem('temaActual');
console.log(tema);

Nota MDN: algunas APIs requieren permisos del usuario y/o contexto seguro (https), como geolocalización, cámara o micrófono.

5Glosario fundamental

Modelo mental recomendado: en JavaScript una variable referencia un valor, y una asignación cambia qué valor referencia ese nombre. Evitamos pensar en "cajas" donde se guardan cosas porque produce errores al razonar sobre el código.

Término Definición breve
ValorDato concreto con el que trabaja el programa (por ejemplo, 42, 'hola', true).
VariableNombre que referencia un valor. En JS hablamos de asignar valores a variables.
AsignaciónOperación de vincular un valor a una variable (por ejemplo, x = 10).
DeclaraciónIntroducción de un identificador en el código, por ejemplo con let, const o function.
Sentencia (Statement)Instrucción completa que ejecuta una acción (declarar, asignar, controlar flujo).
Expresión (Expression)Fragmento de código que produce un valor (por ejemplo, 2 + 2).
OperadorSímbolo o palabra que opera sobre valores (por ejemplo, +, ===, &&).
LiteralValor escrito directamente en el código fuente (por ejemplo, 'Ana', 3.14, {}).
Tipo de datoCategoría de un valor (por ejemplo, string, number, boolean, object).
PrimitivoValor no objeto y de semántica simple (string, number, boolean, null, undefined, symbol, bigint).
ObjetoValor de tipo object formado por pares clave-valor.
ArrayObjeto especial para colecciones ordenadas de valores.
FunciónValor invocable que encapsula una secuencia de instrucciones y puede devolver un resultado.
Parámetro / argumentoParámetro: nombre en la definición de función. Argumento: valor que se pasa al llamarla.
RetornoValor que una función devuelve mediante return.
CondicionalEstructura que decide qué bloque ejecutar según una condición (if, switch).
BucleEstructura de repetición de instrucciones (for, while).
ComentarioTexto que documenta código y no se ejecuta.
JavaScriptLenguaje de programación de propósito general, muy usado en web para interacción y lógica.
ECMAScriptEstándar que define la sintaxis y semántica base del lenguaje.
Motor JSComponente que interpreta/ejecuta JavaScript (V8, SpiderMonkey, JavaScriptCore).
RuntimeEntorno de ejecución con APIs disponibles (navegador, Node.js).
ParsingAnálisis del código fuente para convertir texto en estructura que el motor entiende.
TokenUnidad léxica mínima del código (palabras clave, literales, operadores).
DOMRepresentación en árbol del documento HTML accesible desde JavaScript.
CSSOMRepresentación en árbol de reglas CSS parseadas.
Render TreeCombinación de DOM + CSSOM para calcular qué se dibuja.
LayoutCálculo de tamaños y posiciones de los elementos en pantalla.
PaintPintado final de píxeles en pantalla.
Web APIsAPIs del navegador usadas por JS (DOM, fetch, localStorage, timers).
Type coercionConversión implícita o explícita de un tipo de dato a otro.
ScopeRegión donde una variable existe y es accesible.
