Listas Contenedores: Div y Span

23Listas

  • <ul>: lista desordenada (viñetas)
  • <ol>: lista ordenada (números)
  • <li>: elemento de lista
  • <dl>, <dt>, <dd>: lista de definiciones
  • Las listas pueden anidarse

  • Item A
  • Item B
  • Item C

Ordenada

  2. Segundo
  3. Tercero

24Contenedores: Div y Span

  • <div>: contenedor de bloque (agrupa secciones)
  • <span>: contenedor en línea (dentro de texto)
  • No tienen significado semántico
  • Útiles para aplicar CSS y JavaScript
