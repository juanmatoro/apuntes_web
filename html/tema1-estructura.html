1Estructura básica de HTML

  • Todo documento HTML comienza con <!DOCTYPE html> (declara HTML5)
  • <html>: elemento raíz que contiene todo el documento
  • <head>: metadatos, título, enlaces a CSS/JS (no visible)
  • <body>: contenido visible de la página
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi página</title>
</head>
<body>
    <!-- Contenido aquí -->
</body>
</html>

2Metadatos (head)

  • <meta charset="UTF-8">: codificación de caracteres
  • <meta name="viewport">: responsive en móviles
  • <title>: título en la pestaña del navegador
  • <link>: enlaza CSS externos
  • <script>: enlaza o incluye JavaScript
  • <meta name="description">: SEO
<link rel="stylesheet" href="estilos.css">
<script src="app.js"></script>
